## 엔비디아(NVDA) 지금 사도 될까요?

결론부터 말씀드리면, **"무릎 꿇고 몰빵할 자리"는 아니지만 "추세는 살아있는 우량주"** 입니다. 한 번에 다 사기보다는 **분할로 접근**하시길 권합니다. 아래에 실시간 데이터(2026-06-17 기준)로 근거를 정리했습니다.

---

### 1. 지금 가격 위치 — "고점에서 한 박자 쉬는 중"

| 항목 | 값 |
|---|---|
| 현재가 | **$207.26** |
| 52주 최고가 | $236.54 (현재 고점 대비 **약 -12%**) |
| 52주 최저가 | $142.03 (저점 대비 **+44%**) |
| 50일 이동평균 | $208.6 → **현재가가 살짝 아래** (단기 조정) |
| 150일 / 200일 이동평균 | $191 / $189.5 → **둘 다 위** (장기 추세 견고) |
| 최근 1개월 / 3개월 / 6개월 수익률 | -6.7% / +15% / +17.7% |

**해석:** 장기 추세(200일선)는 우상향으로 멀쩡합니다. 다만 단기적으로는 50일선 아래로 떨어진 **눌림 구간**입니다. 즉, "추세 자체가 깨진 게 아니라 잠깐 숨 고르는" 자리입니다. 이런 자리는 공격적 투자자에겐 분할 매수 기회일 수 있고, 보수적 투자자라면 50일선($208 부근)을 다시 회복하는 걸 확인하고 들어가도 늦지 않습니다.

### 2. 실적 — 이게 핵심입니다. 여전히 폭발적

| 항목 | 값 |
|---|---|
| 최근 분기 매출 | **$81.6B** (전년 동기 $44.1B → **+85% YoY**) |

| 순이익률 | **약 63%** (제조업체로선 비현실적으로 높음) |
| 매출총이익률 | 약 74% |
| ROE | 114% |
| 분기 매출 순차 성장 | 최근 4개 분기 연속 +6% → +22% → +19.5% → +19.8% |

**해석:** 밸류에이션 논쟁은 많아도, **펀더멘털은 여전히 "성장 가속" 국면**입니다. 매출이 1년 만에 거의 두 배, 순이익은 세 배가 됐고, 분기마다 두 자릿수로 더 커지고 있습니다. 이 정도 이익률을 내는 대형주는 시장에 거의 없습니다.

### 3. 밸류에이션 — "비싸 보이지만, 이익 성장이 그걸 빠르게 따라잡는 중"

| 항목 | 값 |
|---|---|
| 후행 PER (Trailing) | 31.7배 |
| 선행 PER (Forward) | **16.3배** |
| 주가매출비율(P/S) | 약 20배 |

**해석:** 후행 PER 31배는 언뜻 비싸 보이지만, 선행 PER이 16배라는 건 **시장이 향후 이익이 다시 한 번 거의 두 배로 늘 것**으로 본다는 뜻입니다. 이익이 이렇게 빠르게 늘면 "비싼 주식"이 시간이 지나며 "합리적인 주식"으로 바뀝니다. 다만 P/S 20배는 절대 싼 수준은 아니므로, **성장이 한 번이라도 꺾이면 밸류에이션 부담이 즉시 주가에 반영**됩니다.
